## Lab Questions

### In plain English describe the function and use of a multiplexer.
The function of a multiplexer is to turn multiple inputs into a single output. 
These ouputs are changed based on the inputs status. Multiplexers can be used for communication via the internent. 
Multiple users or inputs on one cable.
### In plain English describe the function and use of a demultiplexer.
The function of a demultiplexer is to turn one input into multiple outputs. Based on the inputs status a output is chosen. 
Demultiplexer can be used to store information into seperate catagories from one source.
### What other uses might these circuits have? (Think Shannon’s)
